## Description

Rename `products/simple_dashboard/CLAUDE.md.template` to `SKILL.md` and transform it into a self-contained, publicly-safe skill file for the Claude Skills Marketplace.

The current `CLAUDE.md.template` serves as the bot onboarding template for webchat users. It contains:
- Core dashboard-building instructions (interview protocol, industry templates, Chart.js/Tailwind patterns, SPA architecture, i18n) — these must be kept
- Webchat-platform-specific sections (auto-screenshot context, Dashboard Auth Flow with `/api/auth/login` endpoints, DashboardDB CRUD with `/api/data/` server routes, `d{USERID}.wpmix.net` hosting, CSP-constrained `/api/fetch` proxy) — these must be removed
- Internal infrastructure references (ports 8094/8095, IPs 95.217.227.164/62.109.14.209, `/root/` paths) — these must be removed

After transformation, `SKILL.md` becomes the primary source of truth for both:
1. **Skill users**: copy `products/simple_dashboard/` to `~/.claude/skills/simpledashboard/`, Claude discovers via YAML frontmatter, generates `index.html` locally deployable to GitHub Pages/Netlify/Vercel
2. **Webchat onboarding**: `bot.ts` reads `SKILL.md` (path updated in Task 4) instead of `CLAUDE.md.template`

The result must pass a security grep: zero matches for `current_dashboard_screenshot`, `WEBCHAT_INIT`, `Extension sidebar`, `localhost:`, `95.217.`, `62.109.`, `:8094`, `:8091`, `:8095`.
