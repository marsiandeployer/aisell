# ⚡ Быстрый старт Bananzabot

## 🚀 Запуск в 3 шага

### 1. Откройте бота
```
https://t.me/bananza_bot
```

### 2. Отправьте /start

### 3. Опишите свой бизнес (текстом или голосом!)

Например:
```
Я продаю онлайн-курсы по программированию на Python.
Целевая аудитория - начинающие разработчики 20-35 лет.
Бот должен отвечать на вопросы о курсе и записывать на консультацию.
```

## 🎯 Тестирование БЕЗ создания бота

После генерации промпта вы получите ссылку типа:
```
https://t.me/bananza_bot?start=test_123456
```

**Просто перейдите по ней и тестируйте!**

- ✅ Не нужно создавать бота в BotFather
- ✅ Не нужен API ключ
- ✅ Работает сразу
- ✅ Можно поделиться с друзьями/коллегами

## 📲 Создание полноценного бота (опционально)

Если тест понравился, создайте отдельного бота:

1. Откройте @BotFather
2. `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен
5. Отправьте токен в @bananza_bot
6. Готово! 🎉

## 🎤 Голосовые сообщения

Можно надиктовать описание бизнеса голосом:
1. Нажмите на микрофон в Telegram
2. Расскажите о своем бизнесе
3. Отправьте голосовое сообщение
4. Бот автоматически распознает и обработает

## 💡 Примеры описаний

### Хорошо ✅
```
Я консультант по фитнесу. Работаю с женщинами 25-40 лет,
которые хотят похудеть после родов. Бот должен отвечать
на вопросы о программах, собирать контакты и записывать
на первую консультацию.
```

### Плохо ❌
```
Я продаю услуги
```
*(Слишком мало информации)*

## 🔧 Команды

- `/start` - Начать
- `/mybots` - Мои боты
- `/stats` - Статистика
- `/cancel` - Отменить
