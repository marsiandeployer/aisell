## 🎯 КЛЮЧЕВЫЕ ИНСАЙТЫ

### 1. 💰 Монетизация: только 1 из 27 ботов (3.7%)

**Факт:** Только bot_1766885895672 (@master_mataev) имеет **встроенную монетизацию**.

**Цены:**
- Месячная подписка: 499₽
- Разовая подписка: 1999₽
- Личная консультация: 999₽
- Подбор телефона: 1499₽

**Инсайт:** 96.3% создателей НЕ ДУМАЮТ о монетизации при создании бота. Они просто хотят "собрать заявки".

**Рекомендация:**
- Добавить в onboarding вопрос: "Будет ли бот платным или бесплатным?"
- Предложить шаблоны с монетизацией (консультации, подписки, premium контент)

---

### 2. 🔔 Система уведомлений: только 41% используют

**Факт:** Только 11 из 27 ботов используют [NOTIFY_USER] и [NOTIFY_ADMIN].

**Почему это критично:**
- Без уведомлений владелец бота НЕ УЗНАЕТ о новых заявках
- Клиент не получит подтверждение → думает, что заявка не прошла

**Инсайт:** Пользователи **не понимают важность** системы уведомлений.

**Рекомендация:**
- Автоматически добавлять уведомления в промпт при создании бота для бизнеса
- Показать пример в onboarding: "Вот как вы узнаете о новых клиентах"

---

### 3. 📋 Сбор контактов: только 48% собирают

**Факт:** Только 13 из 27 ботов явно собирают контакты (телефон, email).

**Проблема:**
- Без контактов невозможно связаться с клиентом
- Заявка без контактов = бесполезная информация

**Инсайт:** Пользователи думают, что Telegram username достаточно. **Но это не так:**
- Не все пользователи имеют username
- Username может быть заблокирован для незнакомцев

**Рекомендация:**
- В шаблонах ОБЯЗАТЕЛЬНО указывать сбор телефона
- Объяснить в onboarding: "Попросите телефон для связи"

---

### 4. 🎭 Тон общения: путаница с "вежливостью"

**Факты:**
- bot_1766391257462: "Избегайте вежливого общения"
- bot_1766211651062: "Дружелюбный и профессиональный"
- bot_1766885895672: "Понятными и читаемыми"

**Инсайт:** Пользователи **НЕ ЗНАЮТ**, какой тон выбрать для своего бизнеса.

**Рекомендация:**
- В onboarding добавить вопрос: "Какой стиль общения предпочитаете?"
  - Дружелюбный (для салонов, кафе)
  - Профессиональный (для B2B)
  - Прямой/информативный (для техподдержки)

---

### 5. 📝 Списки услуг: все говорят "предоставим список", но НЕ УКАЗЫВАЮТ

**Проблема:**
- Промпты часто содержат: "Бот должен предоставлять информацию об услугах"
- Но **конкретные услуги не перечислены**

**Пример (bot_1766390861018):**
> "Бот может отвечать на вопросы о характеристиках продукта, цене, условиях доставки"

Но КАКОЙ продукт? КАКАЯ цена? **Не указано!**

**Инсайт:** AI не может **придумать** услуги/цены. Нужны **конкретные данные**.

**Рекомендация:**
- В onboarding ОБЯЗАТЕЛЬНО спрашивать:
  - "Перечислите ваши услуги (через запятую)"
  - "Укажите цены (если есть)"
- Добавить примеры для разных ниш:
  - Салон: маникюр (500₽), педикюр (700₽), покрытие гель-лаком (+300₽)
  - Ремонт: диагностика (0₽), замена экрана iPhone (2500₽), чистка от пыли (500₽)

**AI-бот НЕ МОЖЕТ хранить** данные без внешней БД.

**3. "Предварительно подготовленный список FAQ" (bot_1766390861018)**
> "Бот должен иметь предварительно подготовленный список FAQ"

Но список **НЕ ПРИВЕДЕН** в промпте!

**Инсайт:** Пользователи **путают AI-бота с обычным ботом на кнопках**.

## 🏆 ФОРМУЛА ИДЕАЛЬНОГО ПРОМПТА ДЛЯ ЛОКАЛЬНОГО СЕРВИСА

На основе анализа 27 промптов, вот формула **идеального промпта**:

### 1. **Роль и контекст** (обязательно)
```
Ты - Telegram бот для [НАЗВАНИЕ БИЗНЕСА], специализирующегося на [УСЛУГИ].
Твоя цель - принимать заявки от клиентов, отвечать на вопросы и передавать заявки владельцу.
```

### 2. **Список услуг и цен** (обязательно)
```
Мы предоставляем следующие услуги:
1. [УСЛУГА 1] - [ЦЕНА]
2. [УСЛУГА 2] - [ЦЕНА]
3. [УСЛУГА 3] - [ЦЕНА]
```

### 3. **Workflow сбора заявки** (обязательно)
```
Когда клиент хочет записаться, собери:
1. Имя
2. Номер телефона
3. Желаемую услугу
4. Удобное время
```

### 4. **Система уведомлений** (обязательно)
```
После сбора данных:
1. Ответь в чате: "Заявка принята! Мы свяжемся с вами в течение часа."
2. [NOTIFY_USER] Ваша заявка на [УСЛУГА] принята. Мы свяжемся с вами по телефону [ТЕЛЕФОН] для подтверждения.
3. [NOTIFY_ADMIN] Новая заявка от @username:
   • Имя: [ИМЯ]
   • Телефон: [ТЕЛЕФОН]
   • Услуга: [УСЛУГА]
   • Время: [ВРЕМЯ]
```

### 5. **FAQ** (желательно)
```
Частые вопросы:

Q: Где вы находитесь?
A: Мы находимся по адресу [АДРЕС]

Q: Какой у вас график работы?
A: Работаем [ГРАФИК]

Q: Можно ли оплатить картой?
A: Да, принимаем наличные и карты.
```

### 6. **Тон общения** (обязательно)
```
Стиль общения: [ДРУЖЕЛЮБНЫЙ/ПРОФЕССИОНАЛЬНЫЙ/ПРЯМОЙ]
Используй простой язык без сложных терминов.
Без markdown форматирования (звездочек, эмодзи).
```

## 📈 РЕКОМЕНДАЦИИ ДЛЯ УЛУЧШЕНИЯ ПЛАТФОРМЫ

### 1. Интерактивный конструктор промптов

**Проблема:** Пользователи не знают, что включить в промпт.

**Решение:** Пошаговый wizard с вопросами:

```
Шаг 1: Тип бизнеса
□ Салон красоты
□ Ремонт техники
□ Автосервис
□ Другое: ___________

Шаг 2: Перечислите услуги (по одной на строку)
1. ___________
2. ___________
3. ___________

Шаг 3: Укажите цены (опционально)
1. ___________ - _____₽
2. ___________ - _____₽

Шаг 4: Какие данные собирать от клиентов?
☑ Имя
☑ Телефон
□ Email
□ Адрес
□ Другое: ___________

Шаг 5: Куда отправлять уведомления?
○ В личку (мой Telegram ID)
○ В канал/группу: @__________

Шаг 6: Стиль общения
○ Дружелюбный (для салонов, кафе)
○ Профессиональный (для B2B)
○ Прямой (для техподдержки)
```

После заполнения → **автогенерация идеального промпта**.
