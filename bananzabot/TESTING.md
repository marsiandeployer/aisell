## ✅ Как протестировать

### 1. Откройте бота в Telegram
```
https://t.me/bananza_bot
```

### 2. Тестовый сценарий создания бота

#### Шаг 1: Запуск
Отправьте команду:
```
/start
```

**Ожидаемый результат**: Бот отправит приветственное сообщение с объяснением как работать

#### Шаг 2: Описание бизнеса
Отправьте описание вашего бизнеса, например:
```
Я продаю онлайн-курсы по программированию на Python для начинающих. Курс стоит 5000 рублей.
```

**Ожидаемый результат**:
- Бот подтвердит получение описания
- Начнет генерацию промпта (это занимает 5-10 секунд)
- Отправит превью сгенерированного промпта
- Попросит отправить токен от BotFather

#### Шаг 3: Создание тестового бота
1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Введите имя бота (например: "My Test Sales Bot")
4. Введите username бота (должен заканчиваться на "bot", например: "my_test_sales_bot")
5. Скопируйте токен, который вам даст BotFather (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

#### Шаг 4: Отправка токена
Отправьте токен боту @bananza_bot

**Ожидаемый результат**:
- Бот проверит токен
- Создаст бота в базе данных
- Автоматически задеплоит бота (занимает 2-3 секунды)
- Отправит сообщение о успешном деплое

#### Шаг 5: Тестирование созданного бота
1. Найдите вашего нового бота по username (который вы указали в BotFather)
2. Отправьте `/start`
3. Попробуйте задать несколько вопросов о вашем продукте
4. Проверьте, что бот отвечает согласно сгенерированному промпту

### 3. Тестирование команд

#### Команда /mybots
Отправьте:
```
/mybots
```

**Ожидаемый результат**: Список всех ваших созданных ботов с их статусами

#### Команда /stats
Отправьте:
```
/stats
```

**Ожидаемый результат**: Статистика системы (количество ботов, использование памяти, список активных ботов)

#### Команда /cancel
1. Начните создание нового бота через `/start`
2. После ввода описания бизнеса отправьте `/cancel`

**Ожидаемый результат**: Процесс создания бота отменен, можно начать заново

## 🔍 Проверка работы на сервере

### Статус процесса
```bash
pm2 status bananzabot
```

Должен быть статус: **online**

### Логи
```bash
pm2 logs bananzabot --lines 50
```

Должны быть логи:
- `🤖 Bananzabot Constructor started successfully!`
- `[Start] User {id} started conversation`
- `[Deploy] Starting deployment for bot...`
- `[BotInstance] ✅ Bot {name} started successfully`

### Проверка активных ботов
```bash
pm2 logs bananzabot --lines 100 | grep "BotInstance"
```

### База данных ботов
```bash
cat /root/space2/bananzabot/bots_database/bots.json
```

Должен содержать JSON с созданными ботами

### Использование памяти
```bash
pm2 describe bananzabot
```

Проверьте поле "memory" - должно быть разумное значение

## 🐛 Возможные проблемы

### Бот не отвечает
```bash
pm2 restart bananzabot
pm2 logs bananzabot --lines 100
```

### Ошибка при генерации промпта
Проверьте, что OpenAI API ключ валиден:
```bash
cat /root/space2/bananzabot/.env | grep OPENAI_API_KEY
```

### Ошибка при деплое бота
Проверьте логи:
```bash
pm2 logs bananzabot --lines 100 | grep "Deploy"
```

### Созданный бот не работает
1. Проверьте статус через `/stats`
2. Проверьте логи на ошибки:
```bash
pm2 logs bananzabot | grep "error"
```
