# ProjectFlow — Kanban Board

## Что это

Kanban-доска для управления проектами с 4 колонками, таймером задач, календарём и отчётами. 4 страницы.

**Live demo:** https://simpledashboard.wpmix.net/showcases/project-kanban/demo

## Как воспроизвести

Промпт для генерации:

> Сделай Kanban-доску для управления проектами: 4 колонки (Backlog, In Progress, Review, Done), карточки задач с приоритетами и исполнителями, drag-and-drop визуал, таймер задач, список проектов, календарь дедлайнов, отчёты по прогрессу и нагрузке

## Ключевые особенности

- **4 колонки Kanban:** Backlog → In Progress → Review → Done (цветовая индикация)
- **Карточки задач** с приоритетами, исполнителями, дедлайнами
- **Таймер задач** (Courier font, pulse animation для активных)
- **Календарь** дедлайнов
- **Progress bars** по проектам

## Страницы (4)

| Страница | Содержимое |
|----------|-----------|
| tasks | Kanban-доска с 4 колонками, карточки задач, фильтры |
| projects | Список проектов с прогрессом, кол-во задач по статусам |
| calendar | Календарь дедлайнов, upcoming tasks |
| reports | Прогресс по проектам, нагрузка по исполнителям, velocity |

## Бизнес-контекст

Команды разработки, дизайна, маркетинга. Управление задачами в стиле Trello/Jira. Подходит для IT-команд, агентств, стартапов.

## Как адаптировать

- Добавить/убрать колонки (например, Testing, Deployment)
- Кастомизировать приоритеты (Critical, High, Medium, Low)
- Добавить теги, метки, чеклисты внутри карточек
- Подключить GitHub Issues / Jira API
