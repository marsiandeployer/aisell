# LeadTrack — Sales Lead Tracker

## Что это

Трекер лидов и продаж с pipeline-воронкой, управлением контактами и аналитикой. 4 страницы.

**Live demo:** https://simpledashboard.wpmix.net/showcases/lead-tracker/demo

## Как воспроизвести

Промпт для генерации:

> Сделай трекер лидов (lead tracker) для отдела продаж: дашборд с KPI (лиды, конверсия, revenue), таблица лидов с фильтрами по статусу и источнику, pipeline-воронка с этапами (New → Contacted → Qualified → Proposal → Negotiation → Won/Lost), отчёты по менеджерам и каналам

## Ключевые особенности

- **Pipeline-воронка:** New → Contacted → Qualified → Proposal → Negotiation → Won/Lost
- **Таблица лидов** с фильтрами по статусу и источнику
- **KPI:** Total Leads, Conversion Rate, Revenue, Pipeline Value
- **Отчёты:** по менеджерам, каналам, динамика за период

## Страницы (4)

| Страница | Содержимое |
|----------|-----------|
| dashboard | KPI карточки, lead sources (doughnut), pipeline overview (bar), recent activity |
| leads | Таблица лидов с фильтрами (All / New / Contacted / Qualified / Won / Lost) |
| pipeline | Визуализация воронки, конверсии между этапами, avg deal size |
| reports | Revenue over time, leads by source, performance by manager |

## Бизнес-контекст

B2B продажи, отдел продаж. Менеджеры ведут лидов от первого контакта до закрытия сделки. Источники: Organic, Paid Ads, Referral, Social Media, Cold Outreach.

## Как адаптировать

- Заменить этапы воронки под свой процесс
- Добавить кастомные поля (бюджет, дедлайн, приоритет)
- Подключить AmoCRM / HubSpot / Bitrix24
- Изменить источники лидов
