# Funnel Analytics Dashboard

## Что это

Полная аналитика воронки продаж — от рекламных расходов до оплат и депозитов. 4 страницы с разбивкой по UTM, финансовой аналитикой и ROI.

**Live demo:** https://simpledashboard.wpmix.net/showcases/funnel-analytics/demo

## Как воспроизвести

Промпт для генерации:

> Автоматизировать сбор и заполнение аналитики по всей воронке продаж — от рекламы до оплат/депозитов. Расход на рекламу, регистрации, посещения вебинара, выставленные счета, оплаты, депозиты, конверсии, разбивка по UTM-меткам

## Ключевые особенности

- **5-ступенчатая воронка:** Регистрации → Посещения вебинара → Выставленные счета → Оплаты → Депозиты
- **UTM-разбивка:** Яндекс (30-40%), Telegram (15-25%), VK (10-20%), YouTube (8-16%), Instagram (5-10%)
- **Расходы на рекламу** по каналам: РСЯ, VK, Яндекс (12-22K руб/день)
- **ROI расчёт:** расход vs доход с динамикой
- **Суммы оплат:** 998₽, 10200₽, 110000₽ (три тарифа)
- **JIT генерация** на основе реальной структуры Google Sheets

## Страницы (4)

| Страница | Содержимое |
|----------|-----------|
| overview | 4 KPI (регистрации, посещения, оплаты, расход), регистрации по дням, воронка за 7 дней |
| funnel | Визуализация 5 ступеней воронки, метрики по дням с конверсиями |
| utm | Регистрации по каналам (doughnut), оплаты по каналам (bar), детальная таблица |
| finance | Расход vs доход (line), расход по каналам (stacked bar), оплаты по дням |

## Бизнес-контекст

Инфобизнес с вебинарной воронкой. Трафик из Яндекса, Telegram, VK, YouTube, Instagram. Модель: реклама → регистрация → вебинар → счёт → оплата → депозит.

Реальные данные: ~80-130 регистраций/день, конверсия в вебинар 35-50%, оплаты 30-70% от счетов.

## Как адаптировать

- Заменить этапы воронки (например, убрать вебинар для e-commerce)
- Добавить свои рекламные каналы
- Подключить Google Sheets (spreadsheet ID в config.yaml)
- Интеграция с GetCourse API, Яндекс.Директ API
