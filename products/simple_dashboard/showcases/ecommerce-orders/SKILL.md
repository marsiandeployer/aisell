# E-commerce Orders Dashboard

## Что это

Дашборд аналитики интернет-магазина: заказы, топ товары, каналы трафика, возвраты. 4 страницы.

**Live demo:** https://simpledashboard.wpmix.net/showcases/ecommerce-orders/demo

## Как воспроизвести

Промпт для генерации:

> Build an e-commerce analytics dashboard: 30-day orders trend, top products by revenue, traffic channels comparison, inventory alerts and returns analysis. 4 pages: Overview (orders + revenue), Products (top SKUs + inventory), Channels (traffic + conversion), Returns (return rate + reasons).

## Ключевые особенности

- **30-дневный тренд заказов** с недельными паттернами
- **Топ-10 товаров** с выручкой и остатками
- **Каналы трафика** — конверсия по источникам
- **Анализ возвратов** — причины и динамика
- **Цветовая схема:** teal

## Страницы (4)

| Страница | Содержимое |
|----------|-----------|
| Overview | Заказы, выручка, AOV, конверсия KPIs + orders trend + revenue by category |
| Products | Топ SKU, низкий сток, маржа KPIs + horizontal bar + inventory table |
| Channels | Orders by channel bar + conversion rate bar |
| Returns | Return rate KPIs + returns trend + reasons pie |

## Бизнес-контекст

Целевая аудитория: владельцы интернет-магазинов с 5000+ заказов которым нужно видеть что продаётся, откуда трафик и почему возвращают.

## Как адаптировать

- Подключить WooCommerce или Shopify → Google Sheets
- Настроить SKU под свой каталог
- Добавить страницу Customers (LTV, повторные покупки)
