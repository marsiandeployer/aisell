Deployment Guide

1

Copy MCW base files

cp -r /root/MultiCurrencyWallet/build-mainnet/ /var/www/moonpay-wallet/
2

Overlay your config files

cp erc20tokens.js variables.css customStyles.css \
   /var/www/moonpay-wallet/
3

Nginx config

server {
    listen 80;
    server_name wallet.moonpay.example.com;
    root /var/www/moonpay-wallet;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }
}
