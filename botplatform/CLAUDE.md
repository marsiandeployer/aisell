# noxonbot Development Guide

## Project Overview

noxonbot is a Telegram bot that provides AI-assisted development capabilities with a subscription-based onboarding flow.

## Onboarding Testing Script

### Purpose
The auto-test script performs a **FULL END-TO-END ONBOARDING TEST**, simulating exactly what a real user does, including clicking buttons and entering activation codes.

**Location**: `tests/test_onboarding.py`

The test suite performs a **REAL END-TO-END ONBOARDING** with actual user interactions:

## Post-Onboarding Behavior

After completing onboarding, bot switches to AI mode with **auto-claude default**:

### Private Chat (Personal User)
- Write anything without command → **automatically becomes `кл` (Claude)**
- Example: Type `Привет` → runs as `кл Привет`
- Commands still work: `кл`, `ко`, `/кл`, `/ко`

### Group Chat
- Requires explicit command: `кл` or `ко` or `/кл` or `/ко`
- Direct messages without command are ignored

### Implementation
- Code change: Lines 2486-2493 in `bot.ts`
- Check: If private chat AND message has no command → prepend `кл`
- Result: Seamless AI interaction without forcing users to type commands
